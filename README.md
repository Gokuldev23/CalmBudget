# CalmBudget

A calendar-based expense tracker app built with SvelteKit. Track your spending day by day with a clean calendar interface.

## Tech Stack

- **Framework**: SvelteKit + Svelte 5
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Auth**: Better Auth
- **Database**: Drizzle ORM + PostgreSQL

## Developing

Install dependencies and start the dev server:

```sh
pnpm install
pnpm run dev

# or open in browser automatically
pnpm run dev -- --open
```

## Building

```sh
pnpm run build
pnpm run preview
```

> To deploy, install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
